THE RULE Every sport page is SPINE + LENS. The spine is identical everywhere: commitment vs floor · load vs band · session ledger · heatmap. The lens is sport-specific widget sets that mount from evidence, not from settings — if the data tier exists in the logs, the widgets appear; if not, the section collapses. Nobody configures a dashboard.
